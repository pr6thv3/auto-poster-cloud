# Royalty-Free Background Music Assets

Only royalty-free, self-owned, or licensed audio tracks are allowed in this directory. 

## Guidelines

* Do **NOT** commit copyrighted commercial songs under any circumstances.
* Supported track moods include:
  * `suspense`
  * `tech_pulse`
  * `cinematic_riser`
  * `soft_background`
  * `energetic`

If no music track matches the requested mood, the system will fail safely (or skip BGM in mock mode).
