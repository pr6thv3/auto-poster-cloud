# Proof Capture Assets Directory

This directory contains manually recorded or approved video assets used to verify claims visually in "proof" and "payoff" scenes.

## Directory Structure
- `calendar_agent/`: Demo recordings showing automated calendar scheduling or filling.
- `code_refactor_agent/`: Demo recordings showing automated code refactoring or programming assistant runs.
- `ai_tool_demo/`: General AI tools demo clips.
- `before_after_result/`: Visual side-by-side or transition comparisons proving the hook's claim.

## Core Rules
1. **Manual Approval Required**: All proof assets must be manually reviewed and approved before public release.
2. **Private Data Restriction**: Do not include copyrighted user interfaces, credentials, private emails, API keys, or personal calendar schedules. Keep data strictly synthetic or mocked.
3. **Format Guidelines**: 
   - Use short clips (typically 3 to 10 seconds).
   - Ensure the content is vertical-safe (fits within a 9:16 aspect ratio or is crop-safe at 1080x1920).
   - Maintain high contrast and visual clarity for readable phone screens.
